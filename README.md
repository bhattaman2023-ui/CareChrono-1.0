# CareChrono 🩺⏳

CareChrono is an AI-powered symptom timeline tracker designed to help elderly patients and caregivers record symptoms effortlessly and provide doctors with a clear view of symptom progression over time.

## The Problem
Patients often struggle to accurately recall when symptoms started or how they changed, leading to incomplete medical histories and less efficient consultations.

## Our Solution
CareChrono transforms daily symptom logs into an organized timeline and generates concise summaries that healthcare professionals can quickly review.

## Key Features
- 📅 Daily symptom tracking
- 📈 Chronological symptom timeline
- 🎚️ Severity monitoring
- 🤖 AI-generated consultation summaries
- 👵 Elderly-friendly interface
- 🎤 Voice-assisted symptom entry

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- AI: OpenAI API
- Version Control: Git & GitHub
